## Установка зависимостей

Для выполнения **задания 10** требуется библиотека **XGBoost**.

Если библиотека не установлена, выполните установку в командной строке Windows:

```bash
pip install xgboost

После установки перезапустите Jupyter Notebook, чтобы библиотека стала доступна в среде выполнения.
