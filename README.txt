# Домашние задания по курсу машинного обучения (1–10)

В репозитории находятся выполненные задания 1–10 по курсу машинного обучения https://mlcourse.ai/book/index.html.  
Все задания выполнены в формате **Jupyter Notebook**.

Работа выполнялась на компьютере с **Windows 10** с использованием **Anaconda**.  
Jupyter Notebook запускался **через браузер**.  
Исходные задания были получены из Git-репозитория.

---

## 1. Что нужно установить на компьютер

Для того чтобы открыть и запустить код, необходимо установить **только Anaconda**  
(остальное — опционально).

---

### 1.1 Установка Anaconda (обязательно)

1. Перейдите на сайт:
   https://www.anaconda.com/products/distribution

2. Скачайте версию для:
   - **Windows**
   - **Python 3.x**

3. Запустите установочный файл и при установке:
   - нажимайте **Next**
   - выберите **Just Me**
   - путь установки можно оставить по умолчанию
   - галочки можно оставить стандартными

4. Дождитесь окончания установки.

После этого на компьютере появятся:
- Anaconda Navigator
- Jupyter Notebook
- Python
- необходимые базовые библиотеки

---

## 2. Запуск Jupyter Notebook

### Способ 1 (рекомендуемый)

1. Откройте **Anaconda Navigator**
2. Найдите пункт **Jupyter Notebook**
3. Нажмите кнопку **Launch**

Через несколько секунд:
- автоматически откроется **браузер**
- в браузере появится файловый менеджер Jupyter

---

### Способ 2 (через командную строку)

1. Нажмите `Win + R`
2. Введите `cmd` и нажмите Enter
3. В открывшемся окне введите:
   ```bash
   jupyter notebook
4. Нажмите Enter
Jupyter также откроется в браузере.

---

### 1. В интерфейсе Jupyter:
       -перейдите в папку с заданиями
       -щёлкните по нужному файлу .ipynb

### 2. Файл откроется в новой вкладке браузера.

## 4. Как запускать код в ноутбуке

1.Для запуска одной ячейки:

  -нажмите Shift + Enter

2.Для запуска всех ячеек по порядку:

  -в верхнем меню выберите:
   Kernel → Restart & Run All
Результаты вычислений и графики появятся сразу под ячейками.

##5. Установка дополнительных библиотек

-Большинство библиотек уже входят в Anaconda.
-Некоторые библиотеки нужно установить дополнительно.

Все команды ниже вводятся прямо в Jupyter Notebook в отдельной ячейке.

---

### 5.1 Установка Prophet

1.В новой ячейке Jupyter введите:
  !pip install prophet

3.Запустите ячейку (Shift + Enter)

4. После установки перезапустите ядро:
   Kernel → Restart

---

###5.2 Установка XGBoost (используется в задании 10)

1.В новой ячейке Jupyter введите:

  !pip install xgboost

2.Запустите ячейку

3.После успешной установки перезапустите ядро:
  Kernel → Restart

Если после этого код с from xgboost import XGBClassifier выполняется без ошибок — установка прошла успешно.

Если возникает ошибка, выполните установку в командной строке Windows:
```bash
pip install xgboost
